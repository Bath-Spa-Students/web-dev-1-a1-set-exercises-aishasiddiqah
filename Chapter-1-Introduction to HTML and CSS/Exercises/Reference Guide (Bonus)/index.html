Quick Reference Guide
CSS
NO. CSS PROPERTY DESCRIPTION EXAMPLE (INPUT) EXAMPLE (OUTPUT)
1. border Creates a border around the element. 
                    #text {
                        border: solid 3px red;
                    }

This text has a red border.
2. color Changes the color of the text. 
                    #col {
                        color: hotpink; 
                    }

This text is colured pink.
3. background-color Adds color to the background. 
                    #bgColor {
                        background-color: darkorange;
                    }

This text has an orange background.
4. padding Creates space between the content and its border. 
                    #padding {
                        border: solid 3px black;
                        padding: 2px;
                    }

There is a distance between the text and border.
5. text-align This property aligns the text content. 
                    #align {
                        border: 2px solid;
                        padding: 1px;
                        text-align: right;
                    }

This piece of text is on the right side of the box.
6. font-size Changes the size of the font. 
                    #size {
                        font-size: 24px;
                    }

This text is larger than the others.
7. font-weight Controls how thick or thin the text is. 
                    #weight {
                        font-weight: 700;
                    }

This text is thicker than the others.
8. text-decoration This property specifies the decoration added to text. 
                    #decor {
                        text-decoration: overline 2px solid;
                    }

There is a line above this piece of text.
